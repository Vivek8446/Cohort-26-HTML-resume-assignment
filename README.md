## Deployment

1. created new repo
2. pushed index.html to github using following commands,
    - git add .
    - git commit -m "Initial commit"
    - git remote add origin <repo-link>
    - git push -u origin main
3. went to Github Pages for deployment
4. selected main branch and root folder and saved
